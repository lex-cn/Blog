## 重启WSL1

```PowerShell
# PowerShell (admin)
Restart-Service LxssManager

# CMD (admin)
net stop LxssManager
net start LxssManager
```

## WSL1升级为WSL2

```powershell
PS C:\WINDOWS\system32> wsl -l

适用于 Linux 的 Windows 子系统分发版:
Debian (默认)

PS C:\WINDOWS\system32> wsl --set-version Debian 2
```
